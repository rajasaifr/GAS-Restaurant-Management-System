# Restaurant Management System (GAS)
This is a full-stack restaurant management system that includes table reservations, online ordering, payments, and an admin panel. It is built to help restaurants manage their daily operations more easily.

## â­ Features

### User Features
- User login and signup with JWT
- Table reservation with live availability
- Browse menu items
- Place online orders (members get 20% discount)
- View and complete payments
- Membership system for discounts
- Submit feedback and ratings
- Edit profile and change password

### Admin Features
- Dashboard with basic analytics
- Add, update, delete menu items
- Manage staff information
- Manage tables and seating capacity
- View and manage all reservations
- Manage memberships
- View customer feedback
- Manage users and permissions

### System Features
- Real-time table availability
- Role-based access (User/Admin)
- Secure payments
- Reporting and analytics
- Responsive UI with Material-UI
